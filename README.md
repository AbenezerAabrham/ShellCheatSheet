## 🎨 Features

- **✨ Modern Aesthetics**: Built with a premium dark-mode glassmorphism UI.
- **📂 Contextual Organization**: Commands grouped by Navigation, Git, System Info, Networking, Docker, and Vercel.
- **📱 Responsive & Zoom-Ready**: A flexible CSS Grid layout that automatically fills space when you zoom in or out.
- **🚀 Ultra Fast**: 100% static HTML/CSS with zero dependencies.

## 📂 Command Categories

### 1. Navigation & Files
Master the art of moving through the file system and managing artifacts.
- `ls -la`: Hidden files and details.
- `cd [dir]`: Directional navigation.
- `mkdir -p`: Deep directory creation.

### 2. Git Workflow
The essentials for version control and collaboration.
- `git status`: Local state tracking.
- `git add .`: Staging changes.
- `git commit`: Snapshot creation.

### 3. System & Network
Insight into your machine's hardware and connectivity.
- `htop`: Real-time process monitoring.
- `curl -I`: HTTP header inspection.
- `uname -a`: Kernel details.
