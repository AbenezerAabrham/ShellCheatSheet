Shell Command Cheat Sheet

Master the terminal with this essential guide to the most important shell commands.

📂 File Navigation

ls -la List all files including hidden ones with details.
cd [dir] Change directory. Use 'cd ..' to go up.
pwd Print current working directory path.
mkdir -p [dir] Create a new directory (and parents if needed).

🛠️ File Operations

touch [file] Create an empty file.
cp -r [src] [dest] Copy files or directories recursively.
mv [src] [dest] Move or rename files/directories.
rm -rf [path] Force remove files or directories (Careful!).

🖥️ System Info

top / htop Display running processes in real-time.
df -h Show free disk space in human-readable format.
free -m Show memory usage in MB.
uname -a Print system information (kernel version, etc).

🐙 Git Essentials

git status Check the status of your local changes.
git add . Stage all changes for the next commit.
git commit -m "msg" Commit staged changes with a message.
git push origin main Push local commits to the remote repository.

🌐 Network

curl -I [url] Fetch the HTTP headers of a URL.
ping [host] Check connectivity to a remote host.
ssh [user]@[host] Securely login to a remote machine.

🔍 Search & Process

grep "text" [file] Search for text patterns inside files.
tail -f [file] Watch a file grow in real-time (useful for logs).
find . -name "*.js" Find all JS files in the current directory.
