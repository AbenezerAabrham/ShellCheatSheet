Master the Command Line

A comprehensive reference for the most powerful shell commands, organized for efficiency and beauty.

📂 Navigation

ls -laR
List all files, hidden ones, and subdirectories recursively.
cd -
Switch back to the previous working directory.
pushd / popd
Save current directory to a stack and return later.

🛠️ File Ops

cp -rv
Copy files with verbose output to see progress.
rmdir
Remove an empty directory safely.
ln -s [target] [link]
Create a symbolic link (shortcut) to a file.

📝 Text & Logs

grep -rnE "regex"
Powerful recursive search using Extended Regex.
awk '{print $1}'
Extract the first column of data from a file/stream.
sed -i 's/old/new/g'
In-place find and replace text within a file.
sort | uniq -c
Sort lines and count unique occurrences.

🐙 Git Mastery

git checkout -b [name]
Create and switch to a new branch instantly.
git log --oneline --graph
View a visual, condensed history of your project.
git commit --amend
Modify your last commit (useful for fixing typos).
git stash / pop
Temporarily save changes without committing.

🐳 Docker

docker ps -a
List all containers (running and stopped).
docker exec -it [id] bash
Enter a running container's terminal.
docker-compose up -d
Launch multi-container apps in detached mode.
docker system prune
Clean up unused images, containers, and networks.

🚀 Deployment

vercel --prod
Deploy your current directory to production.
vercel logs
Stream live deployment logs to your terminal.
gh repo create
Interactively create a new GitHub repository.

🔒 Permissions

chmod +x [file]
Make a script or file executable.
chown -R [user]:[group]
Change ownership of a folder recursively.
sudo !!
Repeat the last command but with sudo.

⚡ Pro Shortcuts

Ctrl + R
Reverse-search through your command history.
Ctrl + L
Clear the terminal screen instantly.
Tab / Double Tab
Auto-complete commands, paths, and options.
